# Server

Sisal challenge server
## Install packages

```bash
npm i

```
## Run Seeders
```bash
npx prisma db seed

this allows to insert species and subspacies on DB

npx prisma generate

this allows to generate prisma/client

```

## Run server

```bash
npm run start

this allows to run the server

```
